# kitsune-i18n

[![English](https://img.shields.io/badge/lang-English-blue.svg)](#) [![日本語](https://img.shields.io/badge/lang-日本語-red.svg)](README.ja.md)

Internationalization and localization support for Kotlin.

## Features

- Manage translations for multiple languages.
- Dynamic language switching.
- Support for formatted strings.

## Usage

```kotlin
val i18n = I18nManager()
val message = i18n.translate("hello.world")
```
