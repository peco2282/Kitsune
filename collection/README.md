# kitsune-collection

[![English](https://img.shields.io/badge/lang-English-blue.svg)](#) [![日本語](https://img.shields.io/badge/lang-日本語-red.svg)](README.ja.md)

Utilities for collections, iterables, and streams.

## Features

- Extension functions for `Iterable` and `Collection`.
- Enhanced stream operations.
- Custom collectors.

## Usage

```kotlin
val list = listOf(1, 2, 3)
// Use extension functions
```
