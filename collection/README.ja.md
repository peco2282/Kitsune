# kitsune-collection

[![English](https://img.shields.io/badge/lang-English-blue.svg)](README.md) [![日本語](https://img.shields.io/badge/lang-日本語-red.svg)](#)

コレクション、イテラブル、ストリームのユーティリティ。

## 特徴

- `Iterable`と`Collection`の拡張関数。
- 強化されたストリーム操作。
- カスタムコレクター。

## 使い方

```kotlin
val list = listOf(1, 2, 3)
// 拡張関数を使用
```
