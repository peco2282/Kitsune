# kitsune-date

[![English](https://img.shields.io/badge/lang-English-blue.svg)](#) [![日本語](https://img.shields.io/badge/lang-日本語-red.svg)](README.ja.md)

Date and time manipulation and formatting for Kotlin.

## Features

- Simple date formatting.
- Relative time calculations (e.g., "3 days ago").
- Date range utilities.
- LocalDate extensions.

## Usage

```kotlin
val now = LocalDate.now()
val formatted = now.format("yyyy/MM/dd")
```
