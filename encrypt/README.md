# kitsune-encrypt

[![English](https://img.shields.io/badge/lang-English-blue.svg)](#) [![日本語](https://img.shields.io/badge/lang-日本語-red.svg)](README.ja.md)

Multi-platform encryption library for JVM, Android, and Native.

## Modules

- **api**: Core encryption interfaces and JNI support.
- **jvm**: JVM-specific implementations (e.g., AES).
- **android**: Android-specific implementations.

## Features

- AES encryption and decryption.
- Secure key management.
- JNI-based native crypto support.
